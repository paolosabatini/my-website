Reinforcement learning



This was a first approach to reinforcement learning, together with my will of coding in Java for creating a basic game. The idea was to create the game and let it run by using automated inputs in order to make it running automatically and the possibility of being driven by a machine learning algorithm.

The status is that a human-inspired scheduling of action is created to be challanged by the coming algorithm. Track record of the games are saved and stored automatically, so that the training samples are generated, and games are possible to be launched automatically as well.
